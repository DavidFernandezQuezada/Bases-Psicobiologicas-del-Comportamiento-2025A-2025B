# Bases-Psicobiologicas-del-Comportamiento-2025A-2025B
Repositorio académico del curso “Bases Psicobiológicas del Comportamiento”, impartido por el Dr. David Fernández Quezada en la Universidad de Guadalajara (CUCS). Este espacio integra materiales docentes, actividades prácticas, lecturas, y proyectos de divulgación científica elaborados por los estudiantes durante los ciclos 2025A y 2025B.
Propósito del repositorio
Brindar una plataforma abierta para el aprendizaje y la divulgación de la neurobiología del comportamiento, combinando fundamentos teóricos, análisis experimental y comunicación científica.
